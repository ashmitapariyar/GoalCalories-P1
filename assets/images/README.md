# Image Assets

This folder contains all image files used in the application.

## Required Images:

1. **calorie.jpg** - Background image for login and registration pages
   - Recommended size: 1920x1080px
   - Used in: login.html, registration.html

2. **home.webp** - Background image for home page hero section
   - Recommended size: 1920x1080px
   - Used in: home.html

## Adding Images:

If images are missing, add them to this directory:
`c:\xampp\htdocs\GoalCalories-P1\assets\images\`

You can use free stock images from:
- Unsplash.com
- Pexels.com
- Pixabay.com

Search for: "healthy food", "calories", "nutrition", "diet"
